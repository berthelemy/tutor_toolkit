The application is a web application for tutors to manage their students and lessons. Tutors work within the context of one or more schools, and should authenticate to gain access to the application and their data.

The application is built using Django and Bootstrap.

The application is version-controlled using Git.

The application is multi-lingual, currently supporting English and French.

The application is responsive and mobile-first.

The main entities within the application are:
- Schools
    - Tutors
    - Students
- Lessons
    - Date
    - Time
    - Student(s)
    - Objectives
    - Resources
    - Homework
    - Notes
