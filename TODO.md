# Design
- Create a colour scheme for the application
- Create a logo for the application
- Create a favicon for the application
- Decide on a domain name for the application

# Product
- Define the business model
- Define the pricing
- Define the payment process
- Decide on open source license
- Decide whether to operate as multi-school or single-school

# User
- Create a user profile page
- Allow the user to set their default language
- Allow the user to change their first and last names
- Allow the user to select their timezone

# Language
- Place language switcher in the footer of all pages, using simple links for each language, and remove the dropdown menu
- Select the language based on the user's browser language, unless it is overridden in the session or by the user's profile
- Allow the user to override the language in the session

# Schools
- Switching school in the dropdown menu should immediately switch the school and redirect to the dashboard. Users should not need to click a button to switch schools.
- The administrator should be able to add a logo for each school.
- The logo for the selected school should be displayed in the footer of each page.

# Students
- In the Students link, the Name should be a clickable link, taking you to the student's detail page
- The student's detail page should contain a list of the student's lessons, with the objectives and notes for each lesson, and a list of the current unfinished homework.

# Lessons
- When adding a new lesson or editing an existing lesson, the date/time picker should show the time in 5 minute intervals
- Times should be recorded in the user's timezone

# Review by student
- The review by student page should show the objectives and notes for each lesson

# Review by week
- The review by week page should show the objectives and notes for each lesson
- The review by week page should allow the user to select a week
- The review by week page should only show days where there are lessons

# Review by day
- The review by day page should show the objectives and notes for each lesson

# Footer
- The copyright notice should have the current year
